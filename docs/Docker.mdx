## Used ports

- 3306: MariaDB - The port for the database
- 3000: Docusaurus: Port used for the documentation site that you are currently visiting.
- 8000: Frontend: Port used for the website of Open-Leaked.
- 8080: Backend: Port used for the backend of the project. 

:::tip

You can change the required ports if you have a service already running by editing the [docker-compose.yml](#) file

:::
